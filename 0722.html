Linux常用的命令行
Git常用的命令行
  1. command t 新开一个teminal tab
  2. command n 新开一个teminal 窗口
  3. 新建一个目录 mkdir code
  4. 进入这个目录 cd code
  5. 把苍鹭repo克隆下来 git clone https://github.com/Nexuu/cha.git
  6. 将SSH Key私钥加入github settings 实现免密传输文件
  7. 查看当前目录下有哪些文件和目录 ls
  8. 新建一个文件 touch index.html
  9. 在同一窗口 查看一个文件不编辑 cat index.html
  10. 新出一个窗口 查看一个文件不编辑 less index.html
  11. 新出一个窗口 查看一个文件并编辑 vim index.html
  12. 进入文件查看模式 按下 键盘的i键(insert) 进入编辑模式
  13. 推出编辑模式 同时按三个键 esc shift :
  14. q 直接退出不保存 wq 退出并保存
  15. pwd 查看当前所在路径
  16. cd 回到最上层（根）目录
  17. cd .. 回到上级目录
  18. git status 查看当前仓库的状态变更
  19. git add index.html 将某个文件加入暂存区
  20. git add . 将所有文件加入暂存区
  21. git reset 将所有文件从暂存区撤销
  22. git reset index.html 将所有文件从暂存区撤销
  23. git commit -m '备注123' 提交语句
  24. git log 查看提交日志
  25. 可以有很多分支 但是只有一个主干
  26. git push 推送到远程(服务器) 默认master
  27. git push origin master 推送到远程(服务器)的master主干分支
  28. git push origin dev 推送到远程(服务器)dev分支
